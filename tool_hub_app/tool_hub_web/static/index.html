FusionCloudRobot

RY-Robot Tool Hub

统一入口，集中访问路径编辑、任务组装配、任务组混编和任务编辑。

推荐流程 Path Editor -> Task Group Builder / Task Group Mixer -> Task Editor

Path Editor

编辑路径、加载地图，并从当前路径直接生成去程/返程子任务对。

适合从几何路径开始构建任务。

进入工具

Task Group Builder

从单个工作区中选择、排序、改名并导出任务组。

适合整理同一批子任务。

进入工具

Task Group Mixer

导入多个旁路工作区，在独立混编工作区里自由组合任务对。

适合跨任务组混搭重组。

进入工具

Task Editor

编辑任务组中的路点语义字段，并同步更新旁路 workspace 文件。

适合任务组生成后的语义微调。

进入工具

Waypoint Task Builder

通过拖放和树结构编排生成路点任务 XML，并兼容现有 waypoint_tasks 模板目录。

适合维护门禁、电梯、状态切换等任务模板。

进入工具

Task Batch Generator

按目录识别 `n01/n03` 任务模板对，批量生成不同楼层的任务 JSON 文件。

适合基于一套模板快速派生整栋楼任务。

进入工具

Task Attribute Batch Generator

指定参考任务文件与样本属性 XML，批量生成并修正 waypoint task 属性文件。

适合批量派生电梯门控、进出电梯等任务属性模板。

进入工具

Virtual Wall Builder

加载 2D 地图，在网页中绘制虚拟墙，并导出旧 VirtualWallManager 可直接加载的 YAML。

适合脱离 RViz 后维护禁行边界和临时墙。

进入工具
